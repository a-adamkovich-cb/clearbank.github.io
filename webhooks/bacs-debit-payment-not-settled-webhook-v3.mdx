---
title: "Bacs Debit Payment Not Settled webhook"
version: 3
webhook: true
---

**This sends a webhook notification confirming that a Bacs debit payment was not settled.**

#### Request body

```json
{
    "Type": "BacsDebitPaymentNotSettledV3",
    "Version": 3.0,
    "Payload": {...},
    "Nonce":
}
```
