**This sends a webhook notification confirming that a Bacs Mandate initiation has failed.**

#### Request body

```json
{
    "Type": "BacsMandateInitiationFailed",
    "Version": 1,
    "Payload": {...},
    "Nonce":
}
```

#### Payload

|-------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

Length: `1`<br/>Min. Length: `0`<br />        Pattern: `(5¦B¦F¦I¦K¦O)`<br /> <br />
